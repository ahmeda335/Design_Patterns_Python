# 🧬 Prototype Design Pattern – Character Cloning

This project uses the **Prototype Design Pattern** to clone characters for a game or simulation.

## 🔧 Pattern Summary

- The **Prototype Pattern** allows you to copy objects without depending on their classes.
- Useful when object creation is costly or complex.

## 🎮 Character Attributes

- Health
- Strength
- Magic
- Weapon

## 🔁 Clone Modes

- `mode=1`: Reference (no actual copy)
- `mode=2`: Shallow Copy
- `mode=3`: Deep Copy
