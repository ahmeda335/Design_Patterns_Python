# 🖥️ Builder Design Pattern – Computer Types

This project implements the **Builder Pattern** to create custom computer configurations.

## 🔧 Pattern Summary

- The **Builder Pattern** constructs complex objects step-by-step.
- It separates construction logic from the actual object, which allows building different representations.
