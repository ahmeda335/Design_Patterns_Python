# 🚘 Abstract Factory Pattern – Toyota vs BMW Vehicles

This project applies the **Abstract Factory Pattern** to create families of vehicles from different manufacturers.

## 🔧 Pattern Summary

- The **Abstract Factory Pattern** provides an interface to create **related objects** (a family) without specifying their concrete classes.
- Each factory produces a set of products (vehicles) specific to a brand.

## 🏭 Factories & Vehicles

- **ToyotaFactory**
  - ToyotaCar
  - ToyotaTruck
- **BMWFactory**
  - BMWCar
  - BMWTruck

  ---
## 🚀 How to Run

1. Clone or download this repository.
2. Make sure you have Python 3.6+ installed.

python main.py
